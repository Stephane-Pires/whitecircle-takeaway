# Guide de développement

## Pré-commit
- Prettier, oxlint et type-check sont exécutés automatiquement.

## Structure
- Composants : `$lib/components`
- Hooks : `$lib/hooks`
- Schémas : `$lib/schema`
- API : `src/app/api`

## Déploiement
- Vercel est configuré pour déployer automatiquement depuis la branche main.
